CONFIGURAÇÕES DO PROJETO

FRONT-END:

instalar o projeto
 - abrir o terminal no diretório do projeto.
 - rodar o comando npm install

corrigir vulnarabilidades
 - rodar o comando npm audit fix --force

subir o projeto
 - yarn start

subirá na porta 3000


BACK-END:

instalar o projeto
 - abrir o terminal no diretório do projeto.
 - rodar o comando npm install

corrigir vulnarabilidades
 - rodar o comando npm audit fix --force

subir o projeto
 - yarn dev

subirá na porta 5000
